# Theoricus

Beginner steps.

# Installing dependencies

````bash
npm install
````

# Starting app

````bash
make s
````

Then visit [http://localhost:11235](http://localhost:11235)

# Releasing app

````bash
make r
````

# Help

For more options, run:

````bash
make h
````

For more help, please drop a msg at the [project group](https://groups.google.com/group/theoricus)
